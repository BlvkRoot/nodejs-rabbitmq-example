# NODEJS-RABBITMQ-EXAMPLE

In this NODEjs project we wll be creating a simple messaging app using RabbitMQ. :rocket: :rocket:

## Tech Stack
Nodejs, RabbitMQ, Redis, Express, Typescript, Eslint and more.


## Installation
    
```
    yarn install
```

## Running the app

```
    yarn dev
```
